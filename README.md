# SHARP_SAQ_EQ5D_mapping
This page contains the scripts and files for the statistical analyses of the (in)direct mapping of the Severe Asthma Questionnaire (SAQ) to the EuroQol-5D-5L (EQ5D5L).

Study population: Patients living with Severe asthma (Europe)
